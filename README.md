# Django-Tutorial

## For env
```bash
    python3 -m venv myenv
```

## For activate
```bash
    source myenv/bin/activate
```

## For creare a new project
```bash
    (myenv) ubuntu@work:~/django$  django-admin startproject examples
```

## For run server
```bash
    cd examples
    python3 manage.py runserver
```

## For create app
```bash
    python3 manage.py startapp myapp
```
